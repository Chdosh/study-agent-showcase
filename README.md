# Personal Portfolio Showcase

陈东升的 AI 应用开发工程作品集。目前收录 Study Agent、AI Learning Copilot 与 Image to PowerPoint Workflow，重点展示可核验的运行数据、状态设计、异常恢复与测试证据。

## 技术栈

Vite、React、TypeScript、React Router、原生 CSS。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run lint
npm run build
```

在线 Demo 为纯静态产品导览，不接入真实模型，也不收集用户数据。
